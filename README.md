# Operating System Lab Programs

## Overview

This repository contains Operating System lab programs completed as part of the B.Tech Computer Science and Engineering curriculum (KTU).

The programs are written in C and cover fundamental Operating System concepts such as CPU scheduling, memory management, process creation, inter-process communication, file handling, threads, and shared memory.

## Topics Covered

- CPU Scheduling Algorithms
- Process Creation using Fork
- Threads
- Pipes and Inter-Process Communication
- Shared Memory
- File Operations
- Memory Allocation Techniques
- Disk Scheduling Concepts

## Repository Contents

The repository contains programs related to:

- First Come First Serve (FCFS) Scheduling
- Shortest Job First (SJF) Scheduling
- Round Robin Scheduling
- Priority Scheduling
- Banker's Algorithm
- First Fit Memory Allocation
- Best Fit Memory Allocation
- Worst Fit Memory Allocation
- Process Creation using Fork
- Thread Creation
- Pipe Communication
- Shared Memory
- File Handling Operations

## How to Run

Compile a program:

```bash
gcc FileName.c -o output
```

Run the program:

```bash
./output
```

## Purpose

These programs were developed as part of Operating System laboratory work to understand the implementation of core operating system concepts and algorithms.
